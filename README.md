# weather-dashboard-api:-

A Python-based weather dashboard application developed to fetch and process real-time weather information through API integration.

## Features

- Real-time weather reports
- Temperature and humidity tracking
- Wind speed and visibility
- Sunrise and sunset details
- Forecast support
- CSV export
- Error handling

## Technologies Used

- Python
- Requests
- CSV
- REST API

## Output

Example:

Temperature: 39°C  
Humidity: 42%  
Wind Speed: 18 km/h  
Weather: Clear  
Country: India  
Sunrise: 05:42 AM  
Sunset: 06:32 PM  
Tomorrow Forecast: Clear

```bash
python weather_dashboard.py
